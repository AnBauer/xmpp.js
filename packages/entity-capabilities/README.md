# entity-capabilities

[Entity Capabilities](https://xmpp.org/extensions/xep-0115.html) for `@xmpp/client`.

Supports Node.js and browsers.

## Install

```
npm install @xmpp/entity-capabilities
```

## Usage

```js
const caps = require('@xmpp/plugins/entity-capabilities')
```
