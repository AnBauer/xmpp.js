# TCP

TCP transport for `@xmpp/client`.

Included and enabled in `@xmpp/client`.

Supports Node.js.

## Install

```js
npm install @xmpp/tcp
```

## Usage

```js
require('@xmpp/tcp')(client)
client.start('xmpp://example.com:5222')
```
